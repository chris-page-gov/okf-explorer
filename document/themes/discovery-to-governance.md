The architecture that emerges: users express intent; the runtime retrieves current capability descriptions; the model selects tools and binds arguments against formal schemas; the authorisation layer establishes who/what is acting and with which limits; an execution substrate handles retries and compensation; and an evidence layer records provenance, traces, policy decisions and consents. See the [control plane](../../stack/control-plane.md).

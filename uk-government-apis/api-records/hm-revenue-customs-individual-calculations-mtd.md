# Individual Calculations (MTD)

The individual calculations API allows a self-assessment taxpayer via software to: - trigger a self-assessment tax calculation - list all their self-assessment tax calculations for a tax year - retrieve their self-assessment tax calculation result using multiple endpoints The sandbox endpoint is `https://test-api.service.hmrc.gov.uk`.

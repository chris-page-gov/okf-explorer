# NHS login API

Use this API to enable patients and the public to access multiple digital health and social care services with NHS login, a single re-usable login.You benefit from:- using a trusted and secure identity platform- choosing a proper level of end user verification- displaying a brand name the public recognises- meeting the [Identity Verification and Authentication Standard (DAPB3051)](https://digital.nhs.uk/data-and-information/information-standards/governance/latest-activity/standards-and-collections/dapb3051-identity-verification-and-authentication-standard-for-digital-health-and-care-services)For more details, see [NHS login guidance for partners and developers.](https://digital.nhs.uk/services/nhs-login/nhs-login-for-partners-and-developers)

# Safety and Security Import Declarations

The Safety and Security system handles digital communications between: - customs administrators - carriers or their appointed representatives You must provide the UK customs authorities with advance information by submitting an ENS before you bring goods into the UK. The sandbox endpoint is `https://test-api.service.hmrc.gov.uk`.

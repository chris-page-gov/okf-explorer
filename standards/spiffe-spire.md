SPIFFE defines open standards for identifying software systems in dynamic environments; SPIRE issues [workload identities](../glossary/workload-identity.md) after node and workload attestation. A complement to OAuth, not an alternative: workload identity establishes *what* is calling; OAuth establishes *on whose behalf*. Essential for service meshes and agent runtimes.

# Terms
Glossary terms used here: [Attestation](../glossary/attestation.md), [Workload identity](../glossary/workload-identity.md).

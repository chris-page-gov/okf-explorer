# Model

Every work maps to `schema:Legislation`. Core mappings include `legislationIdentifier`, `legislationType`, `legislationDate`, `legislationDateVersion`, `legislationJurisdiction`, `legislationChanges`, `legislationCommences`, `legislationRepeals`, `legislationAmends`, `isPartOf`, `hasPart`, `encoding`, `citation` and `license`.

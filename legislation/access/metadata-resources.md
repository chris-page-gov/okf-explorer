# Guidance

CLML `ukm:Metadata` and AKN `meta` describe identifiers, dates, types, alternatives, associated documents, enabling powers and unapplied changes where available. `/resources/data.xml` provides metadata without the full text. Static PDF and associated-document URIs must be read from metadata.

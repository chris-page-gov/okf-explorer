# Guidance

Research Legislation documents bulk CLML, AKN, HTML and plaintext collections plus core metadata, interpretations, website inventory, in-force and amendments datasets. During this build its anonymous download URLs returned HTTP 401, so the reproducible generator uses year-partitioned Atom feeds instead.

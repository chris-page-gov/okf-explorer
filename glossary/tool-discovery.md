# Definition
Finding **that** a capability exists — via catalogues, [OpenAPI](../standards/openapi.md) docs, [agent cards](agent-card.md) or protocol registries. Distinct from [tool retrieval](tool-retrieval.md), which narrows a large set at inference time.

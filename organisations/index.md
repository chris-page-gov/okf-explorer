# Organisations

Bodies that author or steward the standards, frameworks and policy.

- [Anthropic](anthropic.md) — Introduced MCP (Nov 2024).
- [OpenAI](openai.md) — Function calling, Structured Outputs, Agents SDK & Responses API.
- [Google](google.md) — Launched A2A; TensorFlow Federated; keyboard FL.
- [Linux Foundation](linux-foundation.md) — Hosts A2A, OpenAPI Initiative, CNCF, LF AI & Data.
- [CNCF](cncf.md) — Stewards OPA, OpenTelemetry; confidential-computing community.
- [OpenAPI Initiative](openapi-initiative.md) — Maintains OpenAPI and Arazzo.
- [W3C](w3c.md) — Standards body for GraphQL-era web specs and PROV.
- [IETF / RFC Editor](ietf.md) — Publishes the OAuth, DPoP, mTLS and HTTP-signature RFCs.
- [Government Digital Service (GDS)](gds.md) — Author of the cross-government API standards.
- [DSIT](dsit.md) — Department behind the Blueprint and AI Management Essentials.
- [ICO](ico.md) — UK regulator; AI and data-protection guidance.
- [NCSC](ncsc.md) — Guidelines for secure AI system development.
- [NVIDIA](nvidia.md) — Authors NVIDIA FLARE.
- [Flower Labs](flower-labs.md) — Maintains the Flower federated AI framework.
- [OpenMined](openmined.md) — Open-source privacy-preserving ML community (PySyft).
- [AI Security Institute (AISI)](aisi.md) — UK institute; evidence on agent/MCP tool usage.
